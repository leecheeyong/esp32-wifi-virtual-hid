ESP32 Virtual HID
Overview Flasher GitHub

Wireless USB Keyboard & Mouse

Turn your affordable ESP32-S2 or ESP32-S3 into a driverless, plug-and-play USB HID controller. Control any target device remotely over a standalone Wi-Fi Access Point.

Flash Firmware

02

Connect & Control
